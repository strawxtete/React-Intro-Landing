## React JS: Primeiros Passos

Uma página simples e responsiva desenvolvida para praticar os conceitos iniciais de React.js, componentes e estilização. O projeto apresenta de forma clara o que é a biblioteca e quais são os pré-requisitos necessários para começar a estudá-la.

## Tecnologias e Ferramentas

- React.js (Biblioteca principal)
- JavaScript
- CSS3
- HTML5

## Como executar e visualizar

```
# Clone o repositório
git clone https://github.com/strawxtete/React-Intro-Landing.git

# Entre na pasta
cd React-Intro-Landing

# Instale as dependências
npm install

# Inicie o servidor
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
